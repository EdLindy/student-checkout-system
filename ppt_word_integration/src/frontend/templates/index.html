Help & FAQ

What file formats are supported?

+

We support PowerPoint files (.ppt and .pptx) and Word documents (.doc and .docx).

How should I format my Word document?

+

Your Word document should have speaker notes clearly labeled with "Slide 1", "Slide 2", etc. Each slide's notes should follow its label until the next slide label.

What voice options are available?

+

We offer several voice options, including our default middle-aged American male with an engaging disposition. You can select different voices from the dropdown menu during processing.

Is there a file size limit?

+

Yes, the maximum file size is 50MB for each file (PowerPoint and Word).

How long does processing take?

+

Processing time depends on the size of your files and the number of slides. Typically, it takes 1-3 minutes for a standard presentation.
